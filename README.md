### Steps to run the application:

1. Run Kafka Backend

- Change Directory into kafka backend folder

```
cd kafka-backend/
``` 	
- Install all modules

```
npm install
```
- Run Kafka server

```
npm start
```

<br/>
2. Run REST API Backend

- Change Directory into backend folder

```
cd backend/
``` 	
- Install all modules

```
npm install
```
- Run backend server

```
nodemon index.js
```
<br/>


3. Run React Frontend App

- Change Directory into frontend folder

```
cd frontend/
``` 	
- Install all modules

```
npm install
```
- Run React App server

```
npm start
```
