## Etsy Clone by Parmeet Singh

### Steps to run the application:


1. Run graphQL API Backend

- Change Directory into backend folder

```
cd backend/
``` 	
- Install all modules

```
npm install
```
- Run backend server

```
nodemon index.js
```
<br/>


2. Run React Frontend App

- Change Directory into frontend folder

```
cd frontend/
``` 	
- Install all modules

```
npm install
```
- Run React App server

```
npm start
```
